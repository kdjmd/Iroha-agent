# Iroha Agent v2.1.1 - Windows

这是 v2.1.0 的跨电脑可靠性修正版，重点修复 FullVoice 首次部署、GPT-SoVITS 启动和长期记忆持久化。

## 修复内容

- 修复 Windows `Path` / `PATH` 并存时 GPT-SoVITS 无法创建进程。
- 修复 Numba 在只读运行时目录创建缓存导致语音长期卡住。
- 推理 YAML 使用应用托管可写副本，外部 GPT-SoVITS 与原始配置不被修改。
- 检查 FullVoice 分卷连续性、归档有效性和磁盘空间；使用临时目录部署并保留旧运行时直到验证成功。
- 部署中断、分卷缺失或新版本失败时自动恢复最后可用运行时。
- 自动处理端口占用，健康检查只接受包含 `/tts` 的 GPT-SoVITS OpenAPI。
- 长期记忆使用原子保存、备份恢复、并发保护、去重与上限；一次性任务和密钥不再误记。
- 修复会话右键菜单关闭后被过早释放产生的 `ObjectDisposedException` 和重复弹窗。
- QA 失败写入报告并返回失败码，不再触发 Windows 未处理异常弹窗。

## 使用提示

Portable 不包含约 8 GB 的 GPT-SoVITS 运行时；它可以匹配电脑上已有的运行时。新电脑需要自动部署语音时，请下载全部 FullVoice 分卷并从 `.001` 解压。

发布附件不包含 API Key、聊天记录、长期记忆、用户设置或崩溃日志。

## 验收结果

- Windows 构建、功能、记忆和跨电脑部署回归全部通过。
- 真实 GPT-SoVITS 服务健康检查通过。
- 日语语音生成耗时 6.63 秒，音频时长 4.58 秒，峰值约 `-1.1 dBFS`，播放和临时文件清理通过。
