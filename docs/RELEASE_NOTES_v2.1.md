# Iroha Agent v2.1.0 - Windows

本版本在视觉小说聊天体验基础上，完成 GPT-SoVITS 跨电脑首次启动自动部署，并将仓库与发布流程收敛为 Windows-only。

## 主要更新

- FullVoice 首次启动自动解压完整 GPT-SoVITS 运行时。
- 自动导入彩叶 GSV 权重、参考音频和日语提示文本。
- 自动生成并持久化本机语音配置，后续启动不重复部署。
- 新增实时部署进度动画与失败降级提示。
- 新增设置内“重新部署语音”，保护原始语音包与外部运行时。
- 自动配置默认使用 CPU 全精度，优先保证不同电脑直接可用。
- 新增 Portable 与 FullVoice 1.9 GB 分卷发布格式及 SHA-256 清单。
- 不再构建或发布 Android APK。

## 使用

1. Portable 用户解压 ZIP；FullVoice 用户下载所有分卷并从 `.001` 解压。
2. 运行 `IrohaAgent.exe`。
3. 在设置中填写自己的 DeepSeek API Key。
4. FullVoice 首次启动等待语音部署完成。

## 验证结论

- Windows 编译与独立运行包通过。
- 标准与紧凑窗口主要工作流通过。
- 会话重命名、删除、置顶、设置、记忆和快捷动作通过。
- 微型运行时完整走通自动匹配、7-Zip 部署和安全重新部署。
- 8.2 GB 原始运行时实际解压到全新目录并保存匹配状态。
- 新部署运行时生成 4.74 秒日语 WAV，峰值约 `-1.1 dBFS`，播放与清理通过。

## 已知边界

- 首次语音部署体积大，建议预留至少 20 GB 可用空间。
- CPU 全精度优先兼容性，首次模型启动和单次生成速度取决于处理器。
- API Key 当前保存在当前 Windows 用户的本地 JSON 设置中，尚未接入 DPAPI。
- 公开分享 FullVoice 前必须确认语音和角色素材授权。
