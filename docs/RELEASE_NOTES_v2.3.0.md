# Iroha Agent v2.3.0

发布日期：2026-07-18（2026-07-19 更新安装包）
平台：Windows 10/11 x64

## 本版重点

v2.3.0 将方案 A、B、C 一次性落地为真正可调用的 Tools 与 Skills。模型可以在聊天中搜索网页、检索或保存记忆、计算、创建提醒、读取授权文档、建立私人知识库、查询天气、管理本地日程、保存邮件草稿、分析图片和执行经过确认的轻量系统操作。

## 2026-07-19 界面维护更新

- 修复 125%/150% 高 DPI 下，紧凑窗口错误隐藏语音波形与 GPT-SoVITS 引擎状态的问题。
- 重排语音试听区的播放按钮、双行说明、柔和分隔、波形和本地引擎状态；可用宽度不足时自动安全降级。
- 统一输入区、语音区与底部状态栏的安全间距，避免遮挡、裁切和错位。
- 清除快捷操作按钮底部的二级文字残留，只保留对应功能的图标与主标题。
- 修复能力组合卡片在部分终端出现黑色矩形边缘、系统虚线焦点框的问题。
- 回形针改用透明圆形悬停反馈，并与发送按钮保持至少 8 px 的安全间距。
- 新增完整、紧凑、最小窗口截图，以及 `980×552` 至 `1920×1080` 的 8 组布局矩阵回归。

## 新增能力

- 18 个 Tools，按“陪伴核心、知识与学习、生活助理”三组管理。
- 10 个 Skill 工作方式，可在能力中心单独选择。
- OpenAI Responses、OpenAI/DeepSeek Chat、Anthropic、Gemini 和 Cohere 原生工具协议。
- Brave Search 与无额外 Key 的 Bing RSS 回退；Brave Key 使用 DPAPI 加密。
- DOCX 与文字型 PDF 本地读取；PdfPig 依赖已随便携版打包。
- 本地提醒、日历、ICS 导出、邮件草稿和知识库均采用安全 JSON 存储。
- 回形针选择图片；支持多厂商视觉输入和可选 Tesseract OCR 回退。
- “工具与隐私”能力中心：能力组、联网方式、授权目录、应用白名单和工作方式集中配置。

## 安全边界

- 写入、删除、剪贴板、图片和系统控制每次都需要用户确认。
- 文件工具只能访问授权目录，拒绝目录越界、符号链接和目录联接逃逸。
- 网页工具拒绝本机、私网、云元数据、`file://`、URL 凭据和危险重定向。
- 每次工具对话最多 4 轮，每轮最多 8 次调用；单个结果和网络响应均有限制。
- 邮件草稿只保存在本机，应用不会发送邮件。
- 不包含任意 Shell、PowerShell、任意文件删除、自动登录、支付或发帖能力。

## 已知限制

- 日历为本地日历，可导出 ICS，但尚未接入 Google/Microsoft OAuth 同步。
- 知识库使用本地关键词片段检索，尚未加入向量嵌入。
- 扫描版 PDF 没有文本层时，应使用支持视觉的模型分析页面截图。
- Bing RSS 是免 Key 回退，稳定性和结果结构不如 Brave Search。
- 图片语义理解取决于当前模型是否支持视觉输入。
