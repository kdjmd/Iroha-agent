# Iroha Agent v2.0.0

Windows 主版本完成视觉小说式界面重构，并整合 DeepSeek 对话、本地 GPT-SoVITS 日语语音、长期记忆、提示词压缩、会话管理和自然逐帧动画。

## 使用

1. 下载并解压 `IrohaAgent-Windows-v2.0.zip`。
2. 运行 `IrohaAgent.exe`。
3. 在设置中填写自己的 DeepSeek API Key。
4. 如需语音，确保本机 GPT-SoVITS 服务可在 `127.0.0.1:9880` 访问。

## 验证

- Windows 构建通过。
- 标准、紧凑和设置界面通过截图复核。
- 聊天、会话、设置、记忆和布局功能回归通过。
