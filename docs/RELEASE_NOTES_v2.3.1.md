# Iroha Agent v2.3.1

发布日期：2026-07-21
平台：Windows 10/11 x64

## 本版重点

v2.3.1 是回复体验、GPT-SoVITS 语音与 Unicode 可靠性补丁版。它保留 v2.3.0 的视觉小说界面、18 个 Tools、10 个 Skills、多厂商模型、长期记忆和首次语音自动部署，同时修复长回复阅读、等待反馈、工具轮次、语音假在线及 Emoji 乱码问题。

## 回复与交互

- VN 对白框右下角新增详情入口，长回复可在同风格悬浮窗中滚动阅读完整正文。
- 用户发送后立即显示分阶段加载反馈，覆盖模型连接、工具调用、日语全文核对和 GPT-SoVITS 合成。
- 工具执行达到第 4 轮安全预算后，不再直接报错；系统会停止新增工具调用并根据已完成结果生成最终回复。
- 相同工具与参数只执行一次，减少重复副作用。

## 日语语音

- 中文显示稿与日语语音稿采用全文逐句等价约束，并检查长度、分句与数字覆盖。
- 日语不完整时只补做一次全文翻译；仍不完整则阻止播放，避免用无关短句代替原回复。
- GPT-SoVITS 健康检查加入真实短句合成，不再只根据端口和 OpenAPI 判断可用。
- 本地推理失败时自动恢复服务并重试一次。
- 语音后台不再依赖桌面进程的输出管道，父进程退出后仍可继续推理。
- GPT-SoVITS 输出定向系统空设备，不保存聊天或语音文本日志。

## Unicode 与 Emoji

- 对话、历史、详情、记忆和逐字动画统一按 Unicode 文本元素处理。
- Emoji、肤色、旗帜和 ZWJ 组合不会被拆成孤立 UTF-16 代理项或乱码。
- 自动修复常见 UTF-8/Windows-1252 错误解码文本。
- 朗读前移除不可发音图形，并把 `%`、`+`、`=`、`&` 等常用符号转换为自然日语读法。

## 性能与体积

- 模型与语音 HTTP 连接复用，角色表情层和旧帧索引按需加载。
- 高清背景、立绘、自然表情和角色头像保持原画质。
- 便携包只排除运行时不会读取的旧资源，不通过降低图片质量缩小体积。

## 验收结果

- Windows 主程序构建：通过。
- 标准自动回归：764/764 通过。
- 真实 GPT-SoVITS 回归：10/10 通过。
- 完整多句日语 WAV：生成、响度校验与 Windows SoundPlayer 播放通过。
- 父进程退出后独立 `/tts` 请求：通过。
- 源工程与 D 盘归档回归：结果一致。
- 凭据扫描：未包含 API Key、用户设置、聊天记录、记忆、运行时缓存或日志。
