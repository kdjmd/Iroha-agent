# Changelog

## Unreleased - 2026-07-19

- 修复 125%/150% 高 DPI 下紧凑布局错误隐藏语音波形与 GPT-SoVITS 引擎状态的问题。
- 重排语音试听 Dock 的播放按钮、双行说明、柔和分隔、波形和本地引擎状态，窄窗口自动降级为不拥挤的简版布局。
- 统一完整与紧凑布局的底部安全间距，防止输入区、语音区和状态栏互相遮挡或裁切。
- 移除快捷操作按钮底部的拥挤二级文字，只保留功能图标与主标题。
- 增加 1280×720 紧凑主界面截图与语音 Dock 边界自动验收。

## 2.3.0 - 2026-07-18

- 实现 A、B、C 三组共 18 个 Tools，以及 10 个可选 Skill 工作方式。
- 增加统一工具注册表、JSON Schema 参数校验、风险分级、每次确认、35 秒超时、4 轮/每轮 8 次调用上限和 48 KiB 结果上限。
- 原生适配 OpenAI Responses、OpenAI/DeepSeek Chat、Anthropic Messages、Gemini GenerateContent 和 Cohere v2 Tool Calling。
- 增加 Brave/Bing 联网搜索、网页正文读取和 SSRF 防护；Brave Key 使用 Windows DPAPI 加密。
- 增加显式长期记忆写入/遗忘、计算、日期时间、本地提醒与运行中提醒反馈。
- 增加授权目录文件搜索、TXT/Markdown/CSV/JSON/XML/HTML/DOCX/PDF 读取和本地私人知识库。
- 增加 Open-Meteo 天气、本地日历与 ICS、只保存不发送的邮件草稿、单次剪贴板、系统媒体键和应用白名单。
- 增加回形针图片选择与多厂商原生视觉请求；不支持视觉时安全降级为本地元数据和可选 OCR。
- 增加“工具与隐私”能力中心，统一管理能力组、搜索连接、授权目录、应用白名单和 Skill。
- 引入 PdfPig .NET Framework 运行时，便携版无需外部 PDF 命令即可读取文字型 PDF。
- 增加 61 项 Tools/协议/安全回归和能力中心高 DPI 截图验收。

- 修复 Cohere 模型发现地址，增加设置页高 DPI 截图回归。

## 2.2.0 - 2026-07-17

- 增加厂商优先的模型设置，为 27 个云端与本地模型入口分别保存模型、地址和 Key。
- 适配 OpenAI Responses、OpenAI Chat、Anthropic Messages、Gemini、Cohere 和 Azure OpenAI 基础聊天协议。
- 将“重新部署语音”入口固定放入设置页。

## 2.1.1 - 2026-07-17

- 修复部分 Windows 同时存在 `Path` / `PATH` 时，旧版 .NET 修改子进程环境导致 GPT-SoVITS 无法启动的问题。
- 为 Numba 使用独立可写缓存目录，避免 `librosa` 在只读运行时目录创建缓存时长期阻塞。
- 推理 YAML 自动复制到应用托管目录，GPT-SoVITS 可安全回写，同时保持外部运行时和源配置不变。
- FullVoice 部署改为完整性检查、磁盘空间检查、临时目录解压、就绪标记和失败自动恢复，不再先删除旧运行时。
- 修复分卷缺失、部署中断、跨 Windows 用户路径失效和端口占用时的错误恢复与反馈。
- 健康检查只接受真实 `/tts` OpenAPI，使用无代理短探测和墙钟超时；Python 诊断改为 UTF-8 非缓冲读取。
- 长期记忆改为原子写入、备份恢复、并发保护、去重与数量上限；普通一次性任务和 API 密钥不再误记。
- 修复会话右键菜单关闭时过早释放导致的重复错误弹窗；同一次运行的全局异常提示只显示一次。
- 增加记忆、跨电脑部署、只读配置、缺失分卷、动态端口和真实语音端到端回归。

## 2.1.0 - 2026-07-16

- 增加首次启动 GPT-SoVITS 自动发现、完整运行时解压、模型导入和配置匹配。
- 增加实时语音部署进度浮窗，展示检查、解压、模型导入、启动与完成状态。
- 增加设置内“重新部署语音”，严格限制为应用托管目录，不修改源语音包和外部运行时。
- 语音匹配结果写入本地设置，后续启动直接连接，不重复部署。
- 自动生成的 GPT-SoVITS 配置默认使用 CPU 全精度，提升跨显卡、驱动和无独显电脑的兼容性。
- 服务状态卡与语音控制区显示自动匹配状态。
- 增加部署安全、重新部署、进度 UI 和完整运行时语音 QA。
- 增加 Windows Portable 与 FullVoice 分卷发布脚本和 SHA-256 清单。
- 仓库正式收敛为 Windows-only，不再包含 Android 工程或 APK 发布流程。

- 增加大尺寸角色舞台、VN 对白框、快捷动作和玻璃质感工具栏。
- 增加会话新建、重命名、删除和置顶。

- 改进长期记忆、提示词压缩与角色回复设定。
